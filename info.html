Info

This is information about the various icons you see below this window, as well as information about the interface itself.
The interface has been programmed to appear similar to that of the 1998 Microsoft owned operating system Windows 98.
Below will be described the various icons and what they are.
- Folder Icon. Displays a folder's contents. There is only one on this website. It is an example icon.
- Program Icon. Leads to some favorite games of mine.
- Netscape Navigator. Leads to fun webpages I made!
- Special media Icon. Displays things such as images.
- Text Document Icon. Displays a text file.
- Audio File Icon. Plays audio, such as songs I like!
